# Bluva's Dotfiles

Arch Linux setup with i3, Kitty, Neovim and Polybar.

## Setup

```bash
git clone https://github.com/Bluva/dotfiles.git
```
```bash
cp -r dotfiles/.config/* ~/.config/
```
```bash
cp dotfiles/.bashrc ~/
```
## Notes

- **Window manager:** i3
- **Terminal:** Kitty
- **Editor:** Neovim
